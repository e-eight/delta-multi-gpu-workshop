# Image classification

- Data: CIFAR-10
- Model: ResNet-18
- Cluster: NCSA Delta
- Frameworks: PyTorch, Lightning Fabric

The `src/` directory contains complete Python scripts to finetune ResNet-18 on CIFAR-10. The `src/todo/` directory contains the same files, but with some of the functions undefined -- as an exercise for workshop attendees to implement. The `slurm/` directory contains Slurm scripts to run the Python scripts on NCSA Delta.
